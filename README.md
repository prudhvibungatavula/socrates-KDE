# socrates-KDE

A complete KDE Plasma Wayland rice for Arch/CachyOS.
Inspired by PewDiePie's Hyprland setup but built entirely for KDE Plasma.

## Structure

- `waybar/` — custom Waybar config with scripts for all modules
- `klassy/` — Klassy window decoration config
- `krohnkite/` — KWin tiling script config
- `kde/` — KDE global configs (kwinrc, kdeglobals, shortcuts)
- `themes/` — color schemes and GTK theme settings
- `wallpapers/` — wallpapers used in the setup

## Quick install

```bash
git clone https://github.com/prudhvibungatavula/socrates-KDE
cd socrates-KDE/waybar
./install.sh
```

## What you need

- Arch/CachyOS/Manjaro
- KDE Plasma 6 on Wayland
- See each folder's README for specific install instructions

## Theme stack

- Window decoration: Klassy
- Color scheme: Main (included)
- Icons: Papirus-Dark
- Font: JetBrains Mono Medium 11
- Cursor: capitaine-cursors-light
- Plasma theme: Sweet
- Tiling: Krohnkite
